# Python-GSoC Blogging platform

Blog and management platform for PSF for running GSoC

## Installation

Requires python 3.6+

```bash
pip install -r requirements.txt
```

## Usage

```python
python manage.py runserver 0.0.0.0:8000
```

Default user/pass is 'admin'
